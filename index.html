Data Preprocessing

Now that we have all the neccessary data we will begin to preprocess it. In this step we will be joining all of the dataframes together using the merge function. We will also perform feature and dimension reduction to help us create features that capture the data better and removing features that will become redundant to avoid the Curse of Dimensionality 

        # Removing miliseconds from df1 and df2
        df = df.drop(columns=["milliseconds"])
        df2 = df2.drop(columns=["milliseconds"])
        
        # Renaming time features to be more specific
        df["lap_time"] = df["time"]
        df2["pit_stop_time"] = df2["time"]
        
        df = df.drop(columns=["time"])
        df2 = df2.drop(columns=["time"])
        
        #Joining the dataframes
        joined_df = pd.merge(
            df,
            df2,
            on=['raceId', 'driverId', 'lap'],
            how='outer'
        )
        
        
        array = []
        grouped = joined_df.groupby(['raceId', 'driverId'])
        
        #Access each group to create new features that better fit the data
        for key, group in grouped:
    
            #Take each racer's lap times and find the average
            group['avg_lap_time'] = (group['lap_time'].apply(lambda x: datetime.strptime(x, "%H:%M:%S.%f") if x.count(":") > 1 else datetime.strptime(x, "%M:%S.%f"))).mean().time()
            group = group.dropna()
        
            #Take each racer's pit stop times (if any taken) and find the average
            group['avg_pit_stop_duration'] = (group['duration'].apply(lambda x: datetime.strptime(str(x), "%M.%S.%f") if str(x).count(".") > 1 else datetime.strptime(str(x), "%M:%S.%f") if str(x).count(":") >= 1 else datetime.strptime(str(x), "%S.%f"))).mean()
        
            group['avg_pit_stop_duration'] = group['avg_pit_stop_duration'].apply(lambda x: x.time() if isinstance(x, datetime) else x)
        
            #Record the maximum number of ppit stops
            group['num_of_stops'] = group['stop'].max()
        
            #Remove columns "lap_time" and "duration" now that we're done with them
            group.drop(columns=['lap_time'], inplace=True)
            group.drop(columns=['duration'], inplace=True)
        
            #Record the place of each racer after the race ended
            final_lap = group['lap'].max()
            group['final_position'] = group[group['lap'] == final_lap]['position']
        
            #Drop "lap" since we're done with it
            group.drop(columns=['lap'], inplace=True)
            group = group.dropna()
        
            #Put each grouped dataframe into an array
            array.append(group)
    
        #Reconnect the modified dataframes
        final_df = pd.concat(array).reset_index(drop=True).drop_duplicates()
        display(final_df.head())
